# 1. CursorForge 製品仕様

## 1.1 プロジェクト概要とライセンス
- **ライセンス:** OSS / MIT ライセンス（GitHub公開）。完全無料。
- **ビジネスモデル:** v1.0では収益化スコープ外（GitHub Sponsors等の任意受付のみ）。将来のフリーミアム拡張（Pro機能クローズド化）の余地は確保。
- **EV/OV コードサイニング:** OSS向け無料署名サービス（SignPath.io 等）を第一選択。

## 1.2 サポートOS・動作環境マトリクス (v1.0)
Windows 10 22H2 以降、および Windows 11 全バージョンをターゲットとします。
Win10 22H2未満（1809〜21H2）は互換性検証対象外でありサポートしません。

| 環境 | サポート | 補足 |
|---|---|---|
| Windows 10 22H2 / Win 11 (x64) | ✅ 動作対象 | 主要ターゲット |
| Windows on ARM64 | ✅ 動作対象 | x64 と並行して ARM64 ビルドを提供 |
| 仮想マシン (VMWare / Hyper-V 等) | ✅ 動作対象 | VM Toolsが独自描画する場合はOS設定優先 |
| タブレット / 2-in-1 | ⚠️ 部分対応 | BTマウス接続時のみ。ペン/タッチは非関与 |
| 非英語ロケール / Sandbox | ✅ 動作対象 | 標準機能の範囲内 |
| リモートデスクトップ (RDP/Citrix) | ❌ 対象外 | 起動時に検出・警告ダイアログを表示（描画不完全のため） |
| Windows Server | ❌ 対象外 | 起動時に検出・警告 |

## 1.3 マウスカーソルの基本仕様
- **対応フォーマット:**
  - 新規制作系: `.png`, `.svg`
  - インポート系: `.ico`, `.cur`, `.ani`（`.ani` の新規生成はv1.0スコープ外）
- **高DPI・マルチモニター対応 (6サイズパッキング):**
  - 単一の `.cur` 内に `32x32, 48x48, 64x64, 96x96, 128x128, 256x256px` の6サイズをパッキング。
  - OS側が拡大率に応じて最適な解像度を自動選択します（4K/8K環境やWindows 11のアクセシビリティ「カーソルサイズ」拡大対応のため256pxまで包含）。
- **ホットスポット:**
  - 画像のどの座標をクリック判定とするかの定義。縮小時の丸め誤差を防ぐため、メタデータ上では常に「元画像の絶対ピクセル値（整数）」で保持。
- **ドット絵対応:**
  - 画像インポート時に色数などでドット絵を判定し、ドット絵（Nearest-Neighbor法）か滑らかな画像（Lanczos法）かを自動/手動で切り替え。
- **.ani プレビュー制御:**
  - 静的プレビュー用に最初の1フレームをPNG化、動的プレビュー（クリエイターモード必須）用に全フレームとJIFF情報をRustから提供して再生。

| 役割名 | レジストリ値 | 役割名 | レジストリ値 |

## 1.5 UI・UX設計
### 二面性のあるUI設計
- **一般ユーザー向け:** `.cursorpack` のドラッグ＆ドロップと「適用」のみのシンプル画面。
- **クリエイターモード (編集画面):**
  - **解像度別アサイン方式:** 1役割に対し6サイズ独立して画像割り当てが可能（32pxはドット絵、128pxは滑らか等）。
  - **ホットスポット解像度別設定:** 微細なズレを許容。
  - ファジーマッチング: `wait`, `busy`, `loading` などのファイル名から自動で役割に割り当て。
  - （ペイント機能やベクター編集は外部ツールに委ね、本アプリは管理と適用に専念）

### ライブラリ管理・プレビュー
- **整理機能:** ソート（名前/更新日/適用回数）、検索、お気に入り（スター）。タグやフォルダによる階層分類はv1.0対象外。
- **静的プレビュー:** テーマカード上に17スロットの縮小画像グリッドを表示。ライブプレビュー（OS全体への一時適用）は実装しない。

### 通知 UX (3層分類)
| 層 | 用途 | 実装手段 | イベント例 |
|---|---|---|---|
| 層1: 無通知 | 期待動作の成功 | — | ダークモード自動切替 / 適用成功 |
| 層2: Windowsトースト | 気づきが必要・操作不要 | Win32 COM (`ToastNotificationManager`) | 適用失敗(復元) / 容量警告 / 更新通知 |
| 層3: モーダル | 操作必要・起動阻害 | アプリ内ダイアログ | 重大エラー / 未署名警告 / RDP警告 |

## 1.6 v1.0 MVP リリース基準
**含まれるスコープ:**
- `.cur` 自動生成、`.ani` インポート
- OSサポート (Win10 22H2 / Win11 x64/ARM64)
- トレイ常駐・ダークモード自動切替・メモリ最適化

- パフォーマンス目標値達成、CI検証

- `.ani` 新規生成、ペイント編集
- ライブプレビュー、適用Undo（戻す機能）
- 時刻/アプリフォーカス連動の切替トリガー
- クラウド同期・自動バックアップ
- 他カーソル管理ツールの検知
- Microsoft Store 配布 (最初は.msi)
