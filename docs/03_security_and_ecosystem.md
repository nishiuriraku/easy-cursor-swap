# 3. セキュリティとエコシステム（配布・共有）

## 3.1 .cursorpack パッケージ形式
- 独自Zipベース形式。画像アセット群とメタデータ `theme.json` を格納。
- **theme.json 必須項目:** `schema_version`, `id`(UUID), `name`, `version`(SemVer), `created_at`, `requires_os_shadow`, `cursors`(ロールとパスのマップ)
- **同名インポートの挙動:** `id` が既存テーマと一致した場合、`version` を比較し、上書き / 新規別名保存 / 併存をユーザーに選択させる確認ダイアログを表示。

## 3.2 セキュリティ多層防御
システムへ不正アクセスやクラッシュを引き起こさないための厳格なバリデーションを実施します。

- **Magic Byte (シグネチャ) 検証:** `.cur`, `.png` のファイルの先頭バイトを読み込み、拡張子偽装による不正ファイル（`.exe` など）の混入をブロック。
- **サイズ閾値・Zip爆弾対策:** (設定で上書き可の既定値)
  - `.cursorpack` 圧縮サイズ上限: **50 MB**
  - 展開後合計サイズ上限: **200 MB** (メモリOOMを防ぐため逐次サイズチェック)
  - 個別画像サイズ上限: **10 MB**
  - ユーザー全体保存容量: **1 GB** 超過でトースト警告 (自動削除はしない)
- **パストラバーサル・Symlink防御:** Zip展開パスに `../` が含まれる場合や、シンボリックリンクを含む場合は、システムファイル上書きの危険があるため全体をエラーとして拒否。
- **レジストリ・インジェクション対策:** テーマ名等に含まれるバックスラッシュ(`\`)やNullバイト(`\0`)をサニタイズ。
- **SVGサニタイズ:** `DOMPurify` 相当の処理を行い `<script>`, `<foreignObject>`, `href`, `data:` 等を完全除去（XSS/SSRF防止）。外部URLフェッチ一律拒否。
- **メタデータパージ:** PNG画像内に隠された不正コードやExifなどを取り除き、純粋なピクセルのみを抽出してパッキング。

## 3.3 公式インデックスと提出フロー
独自バックエンドは持たず、GitHub上の単一JSON（公式メタデータインデックス）を Source of Truth とします。

1. **提出プロセス:** クリエイターモードで出力時、ブラウザが開き「Submit Theme」PRテンプレートにURLパラメータ付きで遷移。Git操作不要。
2. **CI自動検証 (GitHub Actions):** 提出されたPRは自動で以下の検証をパスする必要があります。
   - `theme.json` スキーマ・Arrow必須チェック
   - `.cursorpack` のSHA-256ハッシュの整合性
   - Ed25519 署名の検証
   - 50MB サイズ閾値チェック
   - 既知マルウェアハッシュDBとの突合
3. CI通過後、人間（運営）の目視確認を経て公式掲載。

## 3.4 Ed25519 署名運用と公開鍵管理
クリエイターのなりすましやパッケージの改ざんを防ぐための署名フロー。

- **鍵生成と暗号化保存:** クリエイターモードで生成した秘密鍵は、Windows DPAPI (`CryptProtectData`) によりOSユーザーアカウント縛りで暗号化し `~/.custom_cursors/_keys/private.key` に保存。
- **公開鍵登録:** GitHubインデックスリポジトリの `authors/{github_username}.json` に提出。
- **key_id:** 公開鍵の SHA-256 先頭16文字。テーマにはこれのみ記録し、検証時はインデックスからフル公開鍵を引く。
- **警告表示:** 署名なしテーマの初回インポート時、「未署名テーマ」強警告ダイアログを表示。
- **鍵ローテーション:** 過去テーマの再署名は不要。インデックス側で過去鍵 (`historical_keys`) の検証情報も維持。
- **エクスポート/インポート:** PC移行用に Argon2 + ChaCha20Poly1305 等でパスフレーズ暗号化した秘密鍵のエクスポート/インポート機能を提供。

## 3.5 .cursorprofile フルバックアップ
- 個人の環境全体（設定 `config.json` + `~/.custom_cursors/` 配下の全テーマ実体）をZip化。
- OS再インストールやPC移行時のバックアップ（エクスポート）用。
- インポート時はパストラバーサル防御等を通した上でマージ/上書きを選択可能。
- 配布用パッケージ（`.cursorpack`）と明確に区別します。
