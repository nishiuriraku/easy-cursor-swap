UI マップ — ユーザー操作はどこで何を起動するか

各ページに どんな UI 要素 があり、それを操作すると どの composable → どの Tauri IPC が走るかを 1 か所で逆引きする。 docs/architecture.json が Rust モジュール側を網羅するのに対し、本ファイルはフロントエンド側の対になる単一エントリ。 ボタン / トグル / セレクト / 入力 / ドラッグ&ドロップ / キーボードショートカット / モーダルの確定アクションを 1 つずつ列挙する。
