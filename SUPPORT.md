# Getting Support

Thanks for using EasyCursorSwap! This page tells you where to go depending on what you need.

## Quick routing

| If you want to…                    | Go here                                                                                                                                      |
| ---------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| Ask "how do I do X?"               | [GitHub Discussions](https://github.com/nishiuriraku/easy-cursor-swap/discussions)                                                           |
| Report a bug                       | [Open an Issue](https://github.com/nishiuriraku/easy-cursor-swap/issues/new)                                                                 |
| Suggest a feature                  | [Open an Issue](https://github.com/nishiuriraku/easy-cursor-swap/issues/new) — tag it as a discussion if you're unsure                       |
| Report a security vulnerability    | [Private Vulnerability Reporting](https://github.com/nishiuriraku/easy-cursor-swap/security/advisories/new) — see [SECURITY.md](SECURITY.md) |
| Submit a code change               | See [CONTRIBUTING.md](CONTRIBUTING.md)                                                                                                       |
| Submit a theme to the public index | The separate [`easy-cursor-swap-index`](https://github.com/nishiuriraku/easy-cursor-swap-index) repo                                         |
| Read about the architecture        | [`docs/architecture.md`](docs/architecture.md)                                                                                               |

## Before you ask

A few minutes of these checks tend to resolve most issues:

1. **Are you on Windows 10 22H2 or Windows 11?** Other versions are not supported.
2. **Do you have the latest release?** Check the [Releases page](https://github.com/nishiuriraku/easy-cursor-swap/releases).
3. **Have you searched [existing issues](https://github.com/nishiuriraku/easy-cursor-swap/issues?q=is%3Aissue) and [discussions](https://github.com/nishiuriraku/easy-cursor-swap/discussions)?** Someone may have hit the same thing.
4. **Did you read the [README](README.md)?** The installation and "Known Limitations" sections cover the common pitfalls (RDP unsupported, no live preview, `.ani` import-only, etc.).

## When you open an Issue

Include:

- **OS version** (e.g. Windows 11 23H2 x64)
- **EasyCursorSwap version** (Settings → About, or the installer filename)
- **What you tried** and **what happened**, ideally with a screenshot or short clip
- **Logs**, if relevant — they live under `%LOCALAPPDATA%\EasyCursorSwap\logs\` (paths are redacted, but please skim for anything personal before attaching)
- **Crash reports**, if the app crashed — Settings → Logs can list and export them

## Out of scope

This project does **not** provide:

- Per-incident commercial support or a guaranteed response SLA
- Help installing or operating Windows itself
- Custom theme creation services (you can make and share them yourself via the marketplace)
- Support for jailbroken / heavily-modified Windows builds, or non-Windows OSes

## Response time

EasyCursorSwap is maintained by a single person in their spare time. Expect best-effort responses within roughly a week. Security reports are prioritised (see [SECURITY.md](SECURITY.md) for the targeted timeline).

## Conduct

All support interactions are governed by the project [Code of Conduct](CODE_OF_CONDUCT.md). Please be patient and constructive — and the same will come back to you.
